# G.E.M. 邓紫棋 · 动态专辑封面

这是已经整理好的静态网站版本，入口是 index.html。

可发布方式：
- Netlify：把整个 gem-album-canvas-v1 文件夹拖到 Netlify Drop。
- Vercel：导入这个文件夹或仓库，框架选择 Other / Static。
- GitHub Pages：把这个文件夹内容放到仓库根目录，Pages 选择 main 分支根目录。

音频和 MV 后续可以放在：
- assets/audio/专辑目录/
- assets/mv/专辑目录/

网页已预留 mp3/flac/m4a/wav 和 mp4/webm/mov 的接入口。
